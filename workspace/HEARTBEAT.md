# HEARTBEAT.md -- 주기적 체크리스트

> 하트비트 주기마다 아래 항목을 순서대로 확인한다.
> 항목 추가/수정/삭제: 대화에서 "행동 규칙 변경해줘" → rule-editor 스킬 실행.

---

## 체크 항목

- [ ] 대화 중 메모리에 저장할 중요 내용이 발생했는지 확인
  - 사용자가 명시적으로 기억을 요청하지 않았더라도, 중요한 결정/약속/선호가 있었으면 memory-manager로 기록

## 사용자 추가 항목

> rule-editor 스킬을 통해 추가된 주기적 체크 항목이 여기에 기록됨.
> 예: "- [ ] 매일 아침 9시에 읽지 않은 이메일 확인하여 요약 전송"
