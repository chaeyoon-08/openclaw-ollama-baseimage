# MEMORY.md -- 장기 기억 인덱스

> 이 파일은 인덱스 역할만 한다. 각 항목의 상세 내용은 memory/ 폴더의 개별 파일에 기록한다.
> 에이전트는 사용자 요청 시 이 인덱스에서 관련 파일 경로를 찾고, 해당 memory/*.md를 읽어 상세 내용을 확인한다.
>
> 관리 규칙:
>   - 이 파일은 20,000자를 넘지 않도록 유지한다 (OpenClaw 부트스트랩 제한)
>   - 20,000자에 가까워지면 오래된 항목을 memory/archive.md로 이동한다
>   - 각 항목은 한 줄 요약 + memory/ 파일 참조 링크 형식이다
>   - 민감 정보(API 키, 비밀번호)는 기록하지 않는다
>
> 기록 방법: 대화에서 "기억해둬" → memory-manager 스킬 실행.
> 조회 방법: "뭐 기억하고 있어?" → memory-manager 스킬 실행.
> 삭제 방법: "잊어줘" → memory-manager 스킬 실행.
