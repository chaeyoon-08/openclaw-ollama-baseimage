# MODEL_GUIDE.md — 모델 특성 가이드

오케스트레이터가 작업을 서브 에이전트에 위임할 때 어떤 모델에게 맡길지 판단하기 위한 가이드다.
이 파일이 비어 있거나 없으면 아래 절차에 따라 자동 생성한다.

---

## 이 파일이 비어 있을 때

다음 절차를 따른다:

1. `shell_execute`로 `echo $ORCHESTRATOR_MODEL && echo $WORKER_MODELS` 실행 → 현재 모델 목록 확인
2. 각 모델에 대해 웹 검색으로 특성 조사 (파라미터 수, 강점, 약점, 적합 작업 유형)
3. 아래 형식으로 이 파일을 작성한 뒤 저장 (`filesystem` MCP의 `write_file` 도구 사용)
4. 이후 작업 시 이 파일을 참조해 모델을 선택한다
